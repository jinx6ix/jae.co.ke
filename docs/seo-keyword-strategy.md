## 🔑 On-Page SEO Checklist

### For Every Page:
- [ ] Title tag: Primary keyword + secondary keyword
- [ ] Meta description: 150-160 chars with keywords + CTA
- [ ] H1: Include primary keyword
- [ ] H2s: Use variations of target keywords
- [ ] Image alt text: Descriptive with keywords
- [ ] Internal links: 3-5 per page to related content
- [ ] URL: Clean, hyphenated, keyword-containing
- [ ] Schema: FAQ, Review, or Product schema as applicable

### For Blog Posts:
- [ ] Target 1 primary keyword
- [ ] Target 3-5 secondary/long-tail keywords
- [ ] Include keyword in first 100 words
- [ ] Use keyword in first H2
- [ ] Add related posts section
- [ ] Include internal links to service pages
- [ ] Add images with optimized alt text
- [ ] Include table of contents (H2 links)
